Погодное одностраничное веб-приложение.
======================================


Написано на React.js + Redux.

Возможности:
-----------

Добавлять/удалять города

Сохранять локально данные

Автоматически запрашивать погоду по координатам пользователя.
************************************************************

Запуск:
---------------

npm i

npm start
***
